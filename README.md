# 霓虹贪吃蛇 🐍✨

电竞风格的贪吃蛇游戏，采用深色背景、霓虹色彩和发光效果。

## 在线游玩

👉 [立即开始游戏](https://你的GitHub用户名.github.io/snake-game/)

## 功能特性

- 🎮 **完整游戏机制**
  - 方向键/WASD控制
  - 空格键开始/暂停
  - 吃食物增长、碰墙/自己死亡
  - 渐进加速机制

- ✨ **电竞视觉风格**
  - 深色背景 + 动态网格
  - 霓虹渐变标题
  - 蛇身发光渐变色（青色→蓝色）
  - 食物脉冲光晕效果
  - 粒子爆裂特效

- 📱 **响应式设计**
  - 桌面端键盘操控
  - 移动端虚拟方向键
  - 适配不同屏幕尺寸

- 🏆 **进度保存**
  - 最高分本地存储
  - 支持暂停/继续

## 本地运行

直接用浏览器打开 `index.html` 即可。

## 技术栈

- 纯HTML/CSS/JavaScript
- Canvas 2D渲染
- 无需任何依赖
